Suavizacion exponencial

Vamos analizar las ventas de 3 empresas para pronosticar los siguientes 10 aos. Estos datos fueron obtenidos de factset y son series anuales.

Iniciamos de la manera estndar:

rm(list=ls())
setwd("~/Documents/suavizacion")

library(forecast)
library(readxl)
library(tidyverse)

Recordando: * Suavizacin simple es til si la serie de tiempo no tiene estacionalidad ni tendencia, por eso los pronsticos son planos. * Mtodo de holt es til si la serie de tiempo no tiene estacionalidad pero si tiene tendencia, la tendencia es infinita (siempre aumenta o disminuye) * Mtodo de holt amortiguado es til no tiene estacionalidad pero si tendencia pero esa tendencia no es infinita

Vamos a cargar los datos

coca <- read_excel("coca.xlsx")
walmart <- read_excel("walmart.xlsx")
netflix <- read_excel("netflix.xlsx")

Generamos las series de tiempo:

coca <- ts(coca$`Coca-Cola Sales`, start = 1980, frequency = 1)
netflix <- ts(netflix$`Netflix Sales`, start = 2002, frequency = 1)
walmart <- ts(walmart$`Walmart Sales`, start = 1981, frequency = 1)

autoplot(coca)

pronostico_coca <- holt(coca, h=10)

Vamos a comprar diferentes pronostico. Vamos a utilizar ahora el mtodo de holt amortiguado. Para estimar este mtodo se pone como opcin damped = TRUE

pronostico_coca2 <- holt(coca, h=10, level = 0.99, damped = TRUE)

Comparamos los pronsticos, para ocultar el intervalo de confianza en la grfica utilizamos la opcin PI=FALSE.

autoplot(pronostico_coca, PI=FALSE) +
