mayor <- ts(datos$`43 Comercio al por mayor`, start = c(2008,1), frequency = 12)

menor <- ts(datos$`46 Comercio al por menor`, start = c(2008,1), frequency = 12)

Estos indices de ingresos son de INEGI, el año base es 2013=100. Representa los ingresos totales de estas empresas por la venta de bienes.

Necesitamos realizar un analsis grafico para determinar si las series tiene estacionalidad o no. Y dependiendo realizar un pronóstico con suavización exponencial.
