Práctica de suavización exponencial

Iniciamos de la menera estandar:

rm(list=ls())

setwd("~/Documents/suavizacion")

library(tidyverse)

Vamos analizar las ventas de coca-cola, netflix y walmart

ko <- read_excel("coca.xlsx")
wal <- read_excel("walmart.xlsx")
netflix <- read_excel("netflix.xlsx")

ko <- ts(ko$`Coca-Cola Sales`, start = 1980, frequency = 1)
wal <- ts(wal$`Walmart Sales`, start = 1981, frequency = 1)
netflix <- ts(netflix$`Netflix Sales`, start = 2002, frequency = 1)

Hacemos un analisis grafico

Coca-cola

autoplot(ko)

Para la serie de coca-cola, como tiene tendencia, se recomendia el uso del método de holt o holt amortiguado

Realizamos el pronostico de h periodos:

pronostico_ko <- holt(ko, h=10)

pronostico_ko2 <- holt(ko, h=10, damped = TRUE)
